# 📊 Advanced MERN Task Manager

A robust, full-stack Task Management application designed to streamline daily workflows. Built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) and styled with **Tailwind CSS**, this application features a clean separate architecture for client and server sides.

## 🎯 Key Features

* **⚡ Full CRUD Operations:** Create, Read, Update, and Delete tasks seamlessly.
* **📂 Database Persistence:** All tasks are securely stored and managed via MongoDB.
* **🌐 RESTful API:** Robust backend architecture built with Express and Node.js.
* **🎨 Modern Tailwind UI:** Clean, responsive, and minimalist user interface.
* **📱 Cross-Platform Responsive:** Flawless experience on mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (via Mongoose)
